# MenuDice-sample

랜덤으로 메뉴를 보여주고 네이버 지도링크를 제공해주는 토이 프로젝트 https://apps.apple.com/kr/app/menudice/id6445906688

메뉴를 누르면 네이버 지도로 연결되어 해당 메뉴를 검색함
테마 컬러 변경 가능, SAVE버튼 눌러 저장
설정으로 들어가서 메뉴 목록을 삭제, 추가, 초기화가 가능
도움말 모달에서 기능 안내
